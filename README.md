# FriskyTrails CRM

FriskyTrails CRM to manage client leads and assign them to agents.

## Tech Stack

- **Frontend:** React 19, Vite, Tailwind CSS 4
- **Backend:** Node.js (placeholder)

## Getting Started

### Frontend

```bash
cd crm_website_frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```

## Features

- Add and manage client leads
- Assign leads to agents
- Track lead notes and communication logs
- Search, filter, and sort leads
- Dashboard with key metrics
